Guide

설치 가이드

🍎 macOS

  1. 1 DMG 파일을 다운로드합니다.
  2. 2 DMG를 열고 oryangmc.app을 Applications 폴더로 드래그합니다.
  3. 3 처음 실행 시 보안 경고가 나타나면 시스템 설정 → 개인 정보 보호 및 보안 → 무관하게 열기를 클릭합니다.
  4. 4 orya.ng 계정으로 로그인 후 플레이합니다.

🪟 Windows

  1. 1 EXE 파일을 다운로드합니다.
  2. 2 설치 파일을 실행합니다. SmartScreen 경고 시 추가 정보 → 실행을 클릭합니다.
  3. 3 설치가 완료되면 바탕화면 아이콘을 실행합니다.
  4. 4 orya.ng 계정으로 로그인 후 플레이합니다.
